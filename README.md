# N-Triples Parser for Dart

This package lets you parse `.nt` files, aka N-Triples files.

See https://www.w3.org/TR/n-triples/ for more info on N-Triples.

## Getting Started

To use this package, add `n_triples_parser` to your `pubspec.yaml`:

```yaml
dependencies:
  n_triples_parser: ^0.3.0
```
